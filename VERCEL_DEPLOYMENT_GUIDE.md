# Guide de Déploiement sur Vercel

## Étapes pour Déployer le Frontend sur Vercel
1. **Installation de Vercel CLI** :
   ```bash
   npm install -g vercel
   ```

2. **Connexion à Vercel** :
   ```bash
   vercel login
   ```

3. **Déploiement du Projet** :
   Dans le répertoire de votre projet, exécutez :
   ```bash
   vercel deploy
   ```

4. **Configuration des Environnements** :
   - Ajoutez les variables d'environnement nécessaires via le tableau de bord Vercel ou en utilisant la commande :
   ```bash
   vercel env add
   ```

5. **Vérification** :
   Accédez à l'URL fournie après le déploiement pour vérifier que tout fonctionne correctement.
